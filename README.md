### Exercícios de JavaScript Moderno (ES6+)
Este repositório contém uma série de exercícios práticos focados na transição do JavaScript tradicional para o ES6 (ECMAScript 2015). Como Analista e Desenvolvedor, utilizei estes desafios para consolidar conceitos fundamentais que são a base para frameworks modernos como React e Vue.js.

🧠 Conceitos Aplicados:
Programação Orientada a Objetos (POO): Implementação de classes, construtores e métodos para organizar a lógica de negócio (Exemplos: Carro, Poligono).

Herança e Polimorfismo: Uso de extends e super() para reaproveitamento de código entre classes pai e filho (Exemplo: FuncionarioComissionado).

Encapsulamento com Getters: Utilização de get para criar propriedades calculadas de forma elegante.

Funções Modernas: Prática com Funções Anônimas e a evolução para Arrow Functions, otimizando a sintaxe e o escopo do código.

Manipulação do DOM: Interação direta com o navegador para exibição de dados processados.

🛠️ Tecnologias Utilizadas:
HTML5

JavaScript Puro - Padrão ES6+

<i>💡 Por que este repositório é importante?
Dominar a sintaxe ES6 é um divisor de águas para qualquer desenvolvedor. Estes exercícios demonstram não apenas a habilidade de codificação, mas a compreensão de como estruturar sistemas de forma escalável, utilizando as melhores práticas da linguagem antes de avançar para abstrações como o JSX.
