## 📂 Estrutura dos Exercícios: HTML5 & Vanilla JavaScript
Este repositório reúne uma série de exercícios práticos focados na transição do JavaScript tradicional para o ES6 (ECMAScript 2015), desenvolvidos para consolidar os conceitos fundamentais que sustentam frameworks modernos como React e Vue.js. Diferente de projetos que dependem de compiladores ou ambientes complexos, estes desafios utilizam a integração nativa entre HTML e JavaScript, permitindo observar o comportamento real dos motores de renderização diretamente no navegador.

### ⚙️**Destaques Técnicos**:<br>
***Integração Direta***: Uso da tag <script> para execução de lógica ES6 sem a necessidade de transpiladores (como Babel).

***Manipulação de Saída***: Utilização de métodos nativos como document.write e manipulação de elementos para exibição de resultados em tempo real.

***Foco em ES6***: Embora o ambiente seja o HTML clássico, toda a lógica de programação segue os padrões modernos de <kbd>class</kbd>, <kbd>extends</kbd>, <kbd>super</kbd>, <kbd>get</kbd> e <kbd>Arrow Functions<kbd/>.

### 🧠**Conceitos Aplicados**:<br>
***Programação Orientada a Objetos (POO)***: Implementação de classes, construtores e métodos para organizar a lógica de negócio (Exemplos: Carro, Poligono).

***Herança e Polimorfismo***: Uso de extends e super() para reaproveitamento de código entre classes pai e filho (Exemplo: FuncionarioComissionado).

***Encapsulamento com Getters***: Utilização de get para criar propriedades calculadas de forma elegante.

***Funções Modernas***: Prática com Funções Anônimas e a evolução para Arrow Functions, otimizando a sintaxe e o escopo do código.

***Manipulação do DOM***: Interação direta com o navegador para exibição de dados processados.
